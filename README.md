# WebDev_Task1
Java Web Development Task 1 OOP

Необходимо написать объектно-ориентированное приложение согласно требованиям, приведенным ниже. В приложении должна быть реализована функциональность, определенная индивидуальным заданием

Новогодний подарок. Определить иерархию конфет и прочих сладостей. Создать несколько объектов-конфет. Собрать детский подарок с определением его веса. Провести сортировку конфет в подарке на основе одного и нескольких параметров. Найти конфету в подарке, соответствующую заданному диапазону содержания сахара.
